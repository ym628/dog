我们的程序区分了麦轮（麦克纳姆轮）和全向轮两种控制方式，因为麦轮车是4轮车、全向轮车是3轮车。
同时麦轮车下分6种型号分别是：
0：高配麦轮无轴承座
1：高配麦轮摆式悬挂
2：高配麦轮独立悬挂
3：顶配麦轮摆式悬挂常规型
4：顶配麦轮摆式悬挂重载型
5：顶配麦轮独立悬挂常规型
最后还有预留了6：定制专用，注意这个是预留，不起作用的。

全向轮车下分7种型号分别是：
0：高配全向轮三角形极速
1：高配全向轮三角形常规
2：高配全向轮圆形常规
3：高配全向轮圆形重载
4：顶配全向轮重载直径127
5：顶配全向轮重载直径152
6：顶配全向轮重载直径203

下面教大家如何将程序切换为对应产品的程序。以产品【0：高配麦轮无轴承座】为例

①：使用keil5打开工程文件，打开头文件【system.h】。

②：将第35、36行的代码改成如下：
#define Omni 0（全向轮车则为：#define Omni 1）
#define Mec   1（全向轮车则为：#define Mec   0）
这一步则将程序切换为了麦轮车（全向轮车）的程序。

③：重新构建程序并烧录（下载）到单片机上。

④：按下单片机上的复位按键，复位刷新程序。

⑤：调节转接板上左上角的蓝色电位器，使OLED显示屏上左上角的数字显示为【0】（即上面各型号对应的数字，如顶配麦轮摆式悬挂重载型对应4，顶配麦轮独立悬挂常规型对应5）。

⑥：再次按下单片机上的复位按键，复位刷新程序。此时已成功切换为对应产品的程序。

总结一下：修改【system.h】切换麦轮/全向轮程序，调节蓝色电位器切换对应产品型号。
